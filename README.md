# **Конвертер валют**

Этот проект сделал для быстрого конверта определенной суммы из одной валюты в другую

## **Как установить**

Для начало получите ключ - Exchangerate токен, для работы с программой. Как и где это сделать вы найдёте **[тут](https://www.exchangerate-api.com/)** 

Далее в файле с проектом создайте файл .env и вставьте туда
`TOKEN="Ваш Exchangerate API токен`

Python3 должен быть уже установлен. Затем используйте `pip` (или `pip3`, есть конфликт с Python2) для установки зависимостей:

```pip install -r requirements.txt```

Рекомендуется использовать [virtualenv/venv](https://docs.python.org/3/library/venv.html) для изоляции проекта

## **Как запустить программу**

1. Запускаем командную строку (`Win+R >>> CMD`)
2. Перемещаемся в директорию с программой (`cd <путь к файлу>`)
3. Прописываем все нужные аргументы вместе с запуском программы: 
	 `python main.py -b RUB -t EUR -a 10000` где:

		-b <базовая валюта>
		-t <целевая валюта>
		-a <сумма для конверта>

## **Возможные ошибки**

Программа не будет работать с неверным API токеном или неверным курсом валют
